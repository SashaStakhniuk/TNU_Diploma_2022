 base_conf.hФайл базових налаштувань. Містить основні параматри налаштувань для роботи з кубиком. Вміщує список базових типів, корисних макросів для бінарних операцій, та швидке піднесення в цілий степінь
 cube_painters.hБібліотека CubePainers – містить реалізацію всіх анаміційних можливостей кубика
 cube_render.hБібліотека рендера, забезпечуює функціонал для відрисовки відготовленних данних на кубику і реацію на команди WebCtr
 cube_webctr.hБібліотека WebCtr – головної управляючої структури, для взаємодії з сервером, підготовкою даних для анімації (cube_painters), відрисовки (cube_render)
 cubeGL.hБібліотека CubeGL – бібліотека для спрощення рисування та підготовки до відображення анімації на кубику
